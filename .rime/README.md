# .rime

路径：`%APPDATA%\Rime`

官网：[RIME | 中州韵输入法引擎](https://rime.im/)

Rime 是跨平台的输入法软件，Rime 输入方案可通用于以下发行版：

- 【中州韵】 ibus-rime → Linux
- 【小狼毫】 Weasel → Windows
- 【鼠须管】 Squirrel → macOS

取得适合你系统的最新版Rime 输入法，印一份《指南书》，准备开工了！

[Rime 定制指南](https://github.com/rime/home/wiki/CustomizationGuide)
